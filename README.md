# 🦉 Buho Vision - Calendarios de Producción

Sistema de gestión de calendarios para producción de contenido deportivo.

## 📱 Acceso Móvil

- **Calendario Semanal**: [Abrir](./calendar_v3_dashboard.html)
- **Calendario de Producción**: [Abrir](./calendar.html)

## 🎯 Características

- ✅ Gestión de tareas de producción
- ✅ Vista semanal y mensual
- ✅ Asignación de responsables
- ✅ Tracking de estado
- ✅ Exportación a PDF
- ✅ Totalmente responsive (PC, tablet, móvil)

## 💾 Almacenamiento

Los datos se guardan localmente en tu dispositivo usando localStorage. No se pierden al cerrar el navegador.

## 🔄 Sincronización

Para sincronizar datos entre dispositivos, usa la función "Exportar" y "Importar" en el menú de configuración.
